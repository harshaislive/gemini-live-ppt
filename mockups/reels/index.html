Beforest x Portrait Reels

Five ways to turn the deck into cinema.

These are intentionally different art-direction passes for a portrait-first experience. The shared rule is the same across all five: the subtitle system lives inside the image or video canvas, not below it, so the whole screen feels like one immersive story field.

What changes 16:9 stage becomes a 9:16 story frame, captions move onto media, controls become discreet overlays, and motion is handled like a reel instead of a deck.
Shared brand system All concepts use ABC Arizona, the existing Beforest earth palette, and the current slide imagery and language as the raw material.

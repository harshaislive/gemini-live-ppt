# Beforest visual direction

The interface and the agent should remain image-first, spacious, and editorial.

## Design cues
- One dominant image surface at a time.
- Quiet, premium compositions with lots of breathing room.
- Headline-first editorial pacing with minimal chrome.
- Supporting copy should stay short.
- Subtitles should feel stable and elegant, not like debug logs.

## Image behavior
- Use visuals to support the current emotional beat.
- Switch imagery when the conversation moves from tension to proof to place to action.
- Prefer grounded, atmospheric, land-first imagery over generic hospitality shots.
